Choose which output option suits your project, only one option among Console, Console with Regex, Unit Test Results, and Combined Test Results may be chosen at a time. The Regex, Group, and Common Regex Options only affect Console with Regex. Console sets the bug event text to be the full console output, console with regex makes each regex match an individual event, Combuned Unit test results compiles each unit test into one event and Individual Unit test results make each unit test result its own bug event.
